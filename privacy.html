Privacy Policy

This Privacy Policy describes how Masar handles information when operating the Masar automation system.

1. Overview

Masar is a private automation tool designed to generate and publish informational media content. The system operates internally and is not intended as a public platform or user service.

2. Personal Data

Masar does not collect personal user accounts, registrations, or profile data from the public. The system does not maintain databases of personal user information.

3. OAuth Authentication

Masar may use OAuth authentication provided by third-party platforms such as TikTok to authorize content publishing to accounts controlled by the developer. These authentication processes are handled directly by the platform provider.

4. Data Storage

Masar does not store personal information obtained through platform authentication. Any tokens or authorization credentials used for API communication are securely stored and used only for publishing content to authorized accounts.

5. Public Data Sources

The system may process publicly available information from external sources in order to generate informational content. Such information is processed only for the purpose of creating media content.

6. Third-Party Platforms

When Masar interacts with third-party platforms, those platforms may collect information according to their own privacy policies. Masar does not control how those platforms manage their data.

7. Security

Reasonable security measures are implemented to protect authentication credentials and system operations from unauthorized access.

8. Changes to this Policy

This Privacy Policy may be updated periodically to reflect improvements to the system or changes in operational practices.
